# 每周汇总

这个目录用来做每周一次的维护沉淀。

它不是替代日更记录，而是把一周里真正重要的变化收拢成一页，方便后面更新公开总页时快速回看。

## 文件命名

- `YYYY-Www.md`

例如：

- `2026-W23.md`
- `2026-W24.md`

## 每周汇总最少包含什么

- 这一周覆盖的日期范围
- 本周日更记录索引
- 本周公开层已经发生的变化
- 本周候选状态变化
- 本周仍在观察的问题
- 下周优先动作

## 自动生成方式

- 本地可运行：`scripts/create-weekly-summary.sh`
- GitHub Actions 会在每周日北京时间 `10:15` 自动生成当周汇总骨架
- 自动任务只负责起一个汇总页，具体判断和结论仍然需要人工补充
