# Roxi OpenClash 教程：怎么把 Roxi 接进全屋代理

更新日期：2026-06-07

如果你搜的是 `Roxi OpenClash 教程`，真正想解决的问题通常不是概念，而是：

怎么把 Roxi 这条线顺利接进 OpenClash。

## 先说结论

最稳的顺序通常不是一上来就改很多高级项，而是：

1. 先确认 Roxi 单机能跑通
2. 再导入 OpenClash
3. 先用默认规则和默认代理组
4. 最后再做全屋设备验证

## 这条线更适合谁

- 已经会进路由器后台的人
- 想做全屋代理的人
- 已经把 Roxi 在单机上跑通的人

## 为什么不要一上来就折腾很多高级设置

因为 OpenClash 新手最容易卡住的，不是“不会优化”，而是：

- 订阅没导好
- 规则没跑通
- 默认代理组不对
- DNS 冲突

所以先把最小闭环跑通最重要。

## 更省时间的上手顺序

### 1. 先确认 Roxi 单机正常

如果手机或电脑都还没跑通，不建议先跳路由器。

### 2. 导入 OpenClash

先确认：

- 配置文件能不能成功拉取
- 节点列表是不是正常显示
- 代理组是不是自动生成

### 3. 先用默认规则

先验证：

- 国内网站直连
- 国外网站能开
- 常用 App 正常

### 4. 再验证全屋设备

至少要测：

- 手机
- 电脑
- 电视
