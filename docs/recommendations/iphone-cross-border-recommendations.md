## iPhone 场景最该看什么

### 1. VPN 授权是否顺

这是 iPhone 用户最容易忽略的第一步。

### 2. 导入路径是否清楚

很多人不是线路不行，而是导入就卡住了。

### 3. 常用地区是否稳定

更值得先看：

- 日本
- 新加坡
- 香港
- 美国

### 4. 切网络以后是否容易恢复

iPhone 上从 Wi-Fi 切到蜂窝、或反过来，状态恢复能力很重要。

1. 先装推荐客户端
2. 先完成授权
3. 先测一个常用地区
4. 直接测目标网站和 App
