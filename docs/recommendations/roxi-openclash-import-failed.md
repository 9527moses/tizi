# Roxi OpenClash 导入失败怎么办：先查订阅，还是先查配置和默认组

更新日期：2026-06-07

很多人搜 `Roxi OpenClash 导入失败怎么办`，核心问题通常不是不会点，而是：

Roxi 这条线明明能用，为什么一接 OpenClash 就不顺。

## 先说结论

这种情况最常见的原因通常是：

- 订阅链接本身有问题
- 配置文件没有成功拉取
- 节点列表或代理组没有正常生成
- 规则、DNS 或默认组承接异常

所以更适合先查导入链路，再查高级项。

## 最快排查顺序

1. 先确认 Roxi 订阅链接是否完整
2. 再确认配置文件能不能成功拉取
3. 再看节点列表和代理组是否生成
4. 最后才看规则、DNS 和高级设置

## 更该先做什么

### 1. 先确认订阅链接

优先检查：

- 链接是不是完整
- 是否重新复制过
- 是否更新过订阅

### 2. 再确认配置拉取是否成功

重点看：

- 配置文件有没有下载成功
- 节点列表是不是出现
- 代理组有没有生成

### 3. 最后再看默认组、规则和 DNS

只有前两层正常，才值得往这层继续排。

### 4. 单机都没验证过，就直接全屋代理
